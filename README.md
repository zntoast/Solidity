### yml文件配置信息

```
add:
  ip : 127.0.0.1
  port : 37101

contract:
  name : 合约名


account:
  mnemonic : cat node1/data/keys/mnemonic
  address : cat node1/data/keys/address
  publicKey : cat node1/data/keys/public.key
  privateKey : cat node1/data/keys/private.key

```
